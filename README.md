# Speed Typing Game

Welcome to the **Speed Typing Game**! This is a fun and interactive game designed to help you improve your typing speed and accuracy. The game challenges you to type a given passage as quickly as possible, and tracks your performance over time. It is a great way to practice typing for people of all ages.

## Features
- **Typing Challenge**: Test your typing speed with randomly selected passages.
- **Timer**: Track how quickly you can type the passage.
- **Accuracy Calculation**: Measures your typing accuracy and displays it as a percentage.

## Technologies Used
- **HTML**: For structuring the game interface.
- **CSS**: For styling the game elements.
- **JavaScript**: For the logic and interactivity of the game.
  
## Installation

To play the game on your local machine, follow these steps:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Kusumasree789/Speed-Typing-Game.git
2. Navigate into the project directory:
   ```bash
   cd Speed-Typing-Game
3. Open the index.html file in your preferred web browser to start the game.
   ```bash
   open index.html
## How to Play:
1. Once the game loads, a random passage of text will appear on the screen.
2. Begin typing the passage in the input box provided.
3. Your typing speed (words per minute) and accuracy will be calculated as you type.
4. Try to finish typing the passage as quickly and accurately as possible.
5. After finishing, your score will be displayed along with your typing speed and accuracy.
